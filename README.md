# Calculator
# 📘 Project Documentation
Project Title

Simple Calculator

Project Description

This is a simple web-based calculator built using HTML, CSS, and JavaScript in a single file.
It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

Objectives

Perform basic mathematical calculations

Use only one HTML file

No backend or frameworks

Beginner-friendly design

Instant result display

Technologies Used

HTML5 – structure

CSS3 – styling

JavaScript – calculation logic

Features

Addition (+)

Subtraction (−)

Multiplication (×)

Division (÷)

Clear (C) button


⚠️ Limitations

Uses eval() (safe here, but not for large apps)

Only basic arithmetic

No keyboard input

🔮 Future Enhancements
